8.4. Keyboard Shortcuts

The keyboard shortcuts shown here are the defaults. They can be changed through the Pref­er­ences dialog (Section 8.3.17); navigate to the General item's Keys item and make whatever changes you want there.

KeypressActionContext
Ctrl+SpaceInvokes the Content Assist action (Section 8.1.1.8) (code completion)Main window and editable views
Ctrl+* on the numeric keypadExpands all of an editor's folded items; see the Editor view (Section 8.2.6)'s Left Margin Context Menu table for detailsEditor windows
Shift+Ctrl+* on the numeric keypadResets the file's folded structure; see the Editor view (Section 8.2.6)'s Left Margin Context Menu table for detailsEditor windows
Ctrl+,Invokes the Prev­ious An­no­ta­tion action (Section 8.1.1.38) unless a Search view (Section 8.2.15) is active in which case it invokes the Search Previous Match actionAll views
Alt+-Invokes the Show System Menu action (Section 8.1.1.61)All views
Ctrl+.Invokes the Next Annotation action (Section 8.1.1.29) unless a Search view (Section 8.2.15) is active in which case it invokes the Search Next Match actionAll views
Ctrl+/A toggle action that comments or uncomments the current line; see the Editor view (Section 8.2.6)'s Text Area Context Menu table for detailsEditor windows (Perl)
Shift+Ctrl+/Adds a block comment; see the Editor view (Section 8.2.6)'s Text Area Context Menu table for detailsEditor windows (JavaScript)
Ctrl+/ on the numeric keypadA toggle that enables or disables editor folding; see the Editor view (Section 8.2.6)'s Left Margin Context Menu table for detailsEditor windows
Shift+Ctrl+/ on the numeric keypadCollapses all of an editor's foldable items; see the Editor view (Section 8.2.6)'s Left Margin Context Menu table for detailsEditor windows
Shift+Ctrl+\Removes a block comment; see the Editor view (Section 8.2.6)'s Text Area Context Menu table for detailsEditor windows (JavaScript)
DeleteInvokes the Delete action (Section 8.1.1.13)Main window and some views
Alt+EnterInvokes the Properties action (Section 8.1.1.43)Main window and editor views
F2Invokes the Open Declaration action; see the Editor view (Section 8.2.6)'s Text Area Context Menu table for detailsEditor windows
Ctrl+F2Invokes the Terminate action (Section 8.1.1.75)Main window, Squish Test Debugging Perspective (Section 8.1.2.3)
F4Invokes the Open Type Hierarchy action; see the Editor view (Section 8.2.6)'s Text Area Context Menu table for detailsEditor windows
F5Invokes the Refresh action (Section 8.1.1.49) except when the Squish Test Debugging Perspective (Section 8.1.2.3) is open, in which case it invokes the Step Into action (Section 8.1.1.70)Main window
Shift+F5Invokes the Use Step Filters action (Section 8.1.1.82) when debugging or performs syntax validation when a Editor view (Section 8.2.6) is active and editing PerlMain window, Squish Test Debugging Perspective (Section 8.1.2.3) or editor view (Perl)
F6Invokes the Step Over action (Section 8.1.1.71)Main window, Squish Test Debugging Perspective (Section 8.1.2.3)
Ctrl+F6Invokes the Next Editor action (Section 8.1.1.30) and navigates the Editor popupMain window

Ctrl+F7Invokes the Next View action (Section 8.1.1.32)Main window
Shift+Ctrl+F7Invokes the Previous View action (Section 8.1.1.41)Main window
F8Invokes the Resume action (Section 8.1.1.54)Main window
Ctrl+F8Invokes the Next Perspective action (Section 8.1.1.31) and navigates the Perspectives popupMain window
Shift+Ctrl+F8Invokes the Previous Perspective action (Section 8.1.1.40) and navigates the Perspectives popupMain window
Ctrl+F10Invokes the Show View Menu action (Section 8.1.1.63)Views that have a view menu
F12Invokes the Activate Editor action (Section 8.1.1.2)Main window

Ctrl+CInvokes the Copy action (Section 8.1.1.10) unless the Variables view (Section 8.2.21) has focus in which case it performs the Copy Variables actionMain window and views with context menus
Shift+Ctrl+CFor JavaScript, Python, and Tcl, this invokes a toggle action that comments or uncomments the current line. For Perl this keyboard shortcut runs the Perl::Critic. See the Editor view (Section 8.2.6)'s Text Area Context Menu table for detailsEditor windows
Shift+Ctrl+DRuns the Pod::Checker; see the Editor view (Section 8.2.6)'s Text Area Context Menu table for detailsEditor windows (Perl)
Shift+Ctrl+EInvokes the Switch to Editor dialog (Section 8.3.24)Main window
