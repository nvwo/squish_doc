2.50.1. iPhone/iPad-specific

Version 4.0.1.1 is a release that was issued for the iPhone edition only. It contains a couple of fixes that did not make it into the 4.0.1 release.

2.50.1. iPhone/iPad-specific

  • Fixed the touchAndDrag function.

  • Fixed synchronization problems on playback. The waitForObject function sometimes returned before an animation was completed.

  • Added the special key <Return> to be used in the type function for UITextField objects.

  • Record a type(..., "<Return>") statement if the Return key was pressed in a UITextField object.
