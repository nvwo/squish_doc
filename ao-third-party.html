9.2. Third-Party Add-Ons

Various third party add-ons and integrations are available for Squish. Here is a list of the ones known to us.

  • Build Integration/Test Management—CMake/CTest/CDash by KitWare.

    This add-on is designed to provide several Squish-related facilities. One is simply to make Squish findable by CMake. Another is to provide CMake macros that can start the squishserver, run a test with the squishrunner, and parse the resultant test log output. And another is to make use of Squish when configuring and building KWWidgets.

  • Test Management—XStudio integration by XQual (scroll down to find the froglogic ones).

    These launchers (currently for Squish for Qt and Squish for Web) allow Squish to be used by the XStudio test management tool.
