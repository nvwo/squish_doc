Squish 6.0.3 is a maintenance release containing a small number of bug fixes as described below:

2.25.1. Squish IDE-specific

  • Fixed a problem with the First Steps Welcome Page when it is shown during start-up of the IDE

  • Fixed incorrect casing of the testData.dataset function in the completion of the JavaScript editor.

  • Fixed recording of script snippets when using Launch AUT mode.

  • Fixed a problem with the Control Bar not always being shown when executing Test Suites.

  • Properly shutdown creation of Verification Points in case of an unexpected event, like broken communication with the AUT

2.25.2. Scripting

  • Safe invalidation of script references to application objects after an application exit.
