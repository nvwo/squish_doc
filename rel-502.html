Squish 5.0.2 is a maintenance release that provides a small number of critical fixes for the Qt, Web and Android editions.

2.38.1. Qt-specific

  • Fixed hooking problem with 64-bit Windows application that surfaced with Qt 5.0 and later.

2.38.2. Web-specific

  • Restore the ability to test https sites with Firefox 23 or later.

2.38.3. Android-specific

  • Added support for the recent Android 4.3 release.

  • Allow for running multiple tests in parallel.

  • Fixed support for embedded Web views on Windows.

  • Added a resourceName property for better object identification.
