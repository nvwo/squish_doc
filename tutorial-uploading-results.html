4.15.  Tutorial: Uploading Results to Squish Test Center

After running a test, you can analyze the results in the Test Results view (Section 8.2.18). The Squish IDE stores the last 10 test results (configurable from Preferences-Squish -Logging). If you wish to save the result somewhere for further analysis, Squish Test Center provides a very nice HTML report, as well as a convenient way to upload test results to it from the Squish IDE.

The Test Results view

The Upload button is enabled if a Squish Test Center is configured from the Test Center view (Section 8.2.17). Clicking it brings you this dialog:

Uploading Results

You are asked for a Project, Batch, and optional Labels. These fields determine how your result can be organized and grouped with others in Squish Test Center. Now, you should be able to see an entry for this result in the Test Center view (Section 8.2.17).

Test Center View

To see an HTML report of this test result, select it and then click on the toolbar button for Open in Browser. From the browser, you can then drill down into the test report to see the details.
